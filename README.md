- **Run the Project**

      cargo run --bin Cryptocurrency_Simulator -- --genisis

- **Node Communication DTypes**

    - **DTYPE:**
        - **0** - add peer
        - **1** - get list of peers
        - **2** - add transaction
        - **3** - get transaction pool
        - **4** - validate block
        - **5** - get blockchain

    - **Response:**
        - **0** - success
        - **1** - error
        - **2** - valid
        - **3** - invalid
